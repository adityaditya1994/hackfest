# Demo Queries for HR Analytics Chatbot

## Basic Employee Counts
curl -X POST http://localhost:5001/api/chat -H "Content-Type: application/json" -d '{"message": "How many employees do we have?"}'

curl -X POST http://localhost:5001/api/chat -H "Content-Type: application/json" -d '{"message": "How many employees are in OneMind value stream?"}'

curl -X POST http://localhost:5001/api/chat -H "Content-Type: application/json" -d '{"message": "How many active employees do we have?"}'
